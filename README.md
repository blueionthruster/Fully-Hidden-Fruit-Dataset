# Fully-Hidden-Fruit-Dataset
Reflected NIR signal images of fully hidden tomatoes.

Each image has 0, 1, 2, or 3 hidden tomatoes.
The dataset is in Yolo format.

This dataset can be used to train object detection models to locate fully hidden tomatoes.

Please acknowledge Aiden Xu (Florida, USA) when you use the images in this repository.
